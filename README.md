## Development
**Dependencies**
 - webpack
 - HMR
 - babel-loader
 - babel-preset-es2015
 - style-loader
 - css-loader 
 - node-sass
 - sass-loader
 - url-loader
 - jquery
 - prettier


##  Setup

#### Dev
```bash
# install dependencies
npm install

# serve with hot reload at localhost:8000
npm run dev

# build for production with minification
npm run build
```
